# 5. 函数、数组

# 函数

说明：

函数是指一段可以直接被另一段程序或代码引用的程序或代码。函数实现了代码重用

5.1 函数的声明与调用

声明函数有两种：

  • 第一种方式：函数声明（命名函数）
function  函数名( ){
    //函数体; 
}

函数的调用：

  • 第二种方式：函数表达式
let  函数名 = function(){
     //函数体;
}

5.2 函数的参数与返回值

说明：

  1. 由于js中参数不用声明数据类型，所以函数参数可以直接写参数名即可。
  2. 只要函数体内有return语句，那么函数就有返回值。
//第一种方式
function add(num1,num2){
	return num1 + num2;
}
console.log(add(10,20));

//第二种方式
let add = function(num1,num2){
	return num1 + num2;
}
console.log(add(10,20));

5.3 不定参数 arguments（伪数组）

function add(){
	for(let i=0;i<arguments.length;i++){
		document.write(arguments[i]);
	}
}
add(1,2,3,4);

5.4 js 中的作用域

说明：

  • js中变量的作用域，取决于变量声明的位置。

  • 在javaScript中，只有函数作用域，没有块级作用域。

5.4.1 使用var来声明变量：

示例： 
if(true){
	var num1 = 10;
}
function say(){
	var num2 = 20;
}
console.log(num1);     //10   （if块没有独立作用域，所以可以访问）
console.log(num2);     //Uncaught ReferenceError: num2 is not defined

5.4.2 let与var的区别

var num = 10;
if(true){
    var num = 20;
}
console.log(num);  //20

结果为20； 因为 js 没有块级作用域，所以 if 语句中声明的 num 覆盖了第一句代码声明的 num 变量。

为了在javaScript中也能使用块级作用域，ES6中新增了let关键词。使用let声明的变量就存在块级作用域。

if(true){
	let num1 = 10;
}
function say(){
	let num2 = 20;
}
console.log(num1);     //Uncaught ReferenceError: num1 is not defined
console.log(num2);     //Uncaught ReferenceError: num2 is not defined

5.4.3 全局变量

在 javaScript 中，变量也可以不用 let 或 var 声明，直接使用。

示例： 
num = 10;
console.log(num);

全局变量即使是声明在函数中，它的作用域也是全局的
function add(){
	num = 10;
}
add();
console.log(num);

⚠️ 注意:

全局变量尽量不要使用

5.5 立即执行函数

在函数声明后面加上一对小括号，代表此函数立即执行

(function(){
    console.log("立即执行");
})()
说明：

立即执行函数的作用：

  • 在多人开发的项目上，你在全局或局部作用域中声明了一些变量，可能会被其他人不小心用同名的变量给覆盖掉。

  • 可以使用立即执行函数模仿一个函数作用域，用匿名函数作为一个“容器”，“容器”内部可以访问外部的变量，而外部环境不能访问“容器”内部的变量。

  • 所以 (function(){ ... })() 内部定义的变量不会和外部的变量发生冲突，俗称“匿名包裹器”或“命名空间”。

  • jQuery 使用的就是这种方法，将 jQuery 代码包裹在 (function(window,undefined){ ...jquery代码...})(window) 中，在全局作用域中调用 jQuery 代码时，可以达到保护 jQuery 内部变量的作用。

# 数组

数组：是一个值的集合，每个值都有一个索引号，我们称之为下标。下标从0开始，每个下标都对应一个值。

5.6 创建数组

//1、创建数组时不指定长度
let arr = new Array();   
			
//2、创建数组时指定长度（一般不使用）
let arr = new Array(5);  
			
//3、创建数组时直接赋值
let arr = new Array(10,20,30);   
			
//4、创建数组时直接赋值
let arr = [66,80,90,77,59];

javaScript中的数组有如下特点：

  1. 数组下标从0开始。
  2. 虽然创建数组时，指定了长度，但实际上数组都是可变长度的，即使超出指定长度也不会有任何问题。
  3. 数组存储的数据可以是任何类型。

5.7 遍历数组

数组中有一个属性length，此属性返回数组的长度

let arr = [55,23,89,65,11];  
console.log(arr.length);        //5
			
//遍历数组
for(let i=0;i<arr.length;i++){
	console.log(arr[i]);
}

5.8 冒泡排序

排序算法有很多：插入排序、选择排序、快速排序、桶排序、冒泡排序等等。

冒泡排序

let arr = [1, 2, 3, 4, 5]
//第一个for循环控制循环次数
for(let i = 0; i < arr.length - 1; i++){
    //第二个for循环控制比较次数
    for(let j = arr.length - 1; j > i; j--){
        if(arr[j] < arr[j-1]){
            let temp = arr[j];
		    arr[j] = arr[j-1];
		    arr[j-1] = temp;
		}
	}
}
//遍历数组并输出
for(let i=0;i<arr.length;i++){
    console.log(arr[i]);
}
//5 4 3 2 1

5.9 多维数组

如果一维数组中的每一个元素也是一个数组，那么就是二维数组。以此类推，还可以有多维数组。

二维数组实例：

示例 
let arr = [
	['HTML','CSS','javaScript'],
	['java','servlet','spring'],
	['Vue','React','Angular']
];
//遍历二维数组
for(let i=0;i<arr.length;i++){
	for(let j=0;j<arr[i].length;j++){
	    document.write(arr[i][j]+' ');
	}
	document.write('<br>');
}

# 6. javaScript内置对象

# 6.1 Array 数组内置对象

类别 名称 描述
属性 length 设置或返回数组中元素的数目
push(el) 将新元素添加到一个数组的尾部, 并返回数组的新长度值。
unshift(el) 将新元素添加到一个数组的尾部, 并返回数组的新长度值。
pop() 移除数组中的最后一个元素并返回该元素。
shift() 移除数组中的第一个元素并返回该元素。
方法 sort() 对数组排序 --不传参时，若数组的元素为字符串，则按元素项首字母进行排序，如['a', 'b', 'ab' ]，排序后为['a', 'ab', 'b']。数字同理。
reverse() 对数组进行反转（注意：不是倒序）
concat(el) 返回一个新数组,这个新数组是由多个数组组合而成的。
join(str) 把数组的所有元素放入一个字符串中,通过一个指定的分隔符进行分隔。
split(str) 将一个字符串分割成一个数组。
splice(...) 可以对数组元素做删除、插入等多种操作。 --改变原数组
slice(...) 返回一个数组的片段。 --不改变原数组

  1. splice() 方法

# 6.3 Date 日期内置对象

示例: 
let mydate = new Date();
console.log(mydate);  //Tue Aug 10 2022 17:15:22 GMT+0800 (中国标准时间)

获取当前日期：

function getCurDate() {
	let now = new Date();
	let year = now.getFullYear();
	let month = now.getMonth() + 1;
	let day = now.getDate();
	month = month < 10 ? "0" + month : month;
	day = day < 10 ? "0" + day : day;
	return year + "-" + month + "-" + day;
}
console.log(getCurDate());

# 6.5 Global内置对象

Global(全局)对象是ECMAScript中一个特别的对象，此对象不可直接访问。在ECMAScript中，不属于任何其他对象的属性和方法，都属于Global。

示例：

  1. parseInt() 将字符串转换成整形

    let str = '100';
console.log(parseInt(str)+1);       //101

  2. parseFloat() 将字符串转换成浮点型

    let str = '100.5';
console.log(parseFloat(str)+1);       //101.5

  3. eval() 将一个字符串解析为javascript代码并执行

    console.log(eval('1+2'));            //3

  4. isNaN() 判断是不是 不是一个数字

    isNaN("aaa");         //返回值为true
