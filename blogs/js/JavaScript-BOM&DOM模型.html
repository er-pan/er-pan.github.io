# 7. BOM模型

# 7.1 BOM模型与DOM模型简介

说明

  1. 浏览器对象模型BOM（Browser Object Model）： 提供了独立于内容的、可以与浏览器窗口进行互动的对象结构。
  2. 文档对象模型DOM（Document Object Model）： 定义访问和处理HTML文档的标准方法。DOM 将HTML文档呈现为带有元素、属性和文本的树结构（节点树）。

# 7.2 BOM编程

# 7.2.1 window对象

window对象是BOM的核心，window对象指当前的浏览器窗口。 window对象是整个浏览器对象模型中的顶层对象，它下有很多子对象（包括document对象（DOM））

# 7.2.2 alert()对话框

alert()方法：在页面上弹出一个只有确定按钮的对话框

window作为顶层对象，可以省略

示例： 
window.alert('hello')
alert('hello')

# 7.2.3 confirm()对话框

confirm()方法：在页面上弹出一个有确定和取消按钮的对话框。

示例： 
if(confirm('确定要删除吗')){
	console.log('删除');
}else{
	console.log('不删除');
}

# 7.2.4 prompt输入框

prompt()方法：在页面上弹出一个有确定和取消按钮的对话框，允许用户输入数据并返回。

示例： 
let score = prompt('请输入成绩：');
console.log(score);

# 7.2.5 open()与close()方法

  • open()方法：打开一个浏览器窗口。
  • close()方法：关闭一个浏览器窗口。
示例： 
<input type="button" value="打开百度" onclick="openBaidu()">
<input type="button" value="关闭百度" onclick="closeBaidu()">
    
<script>
	//声明一个引用
	let obj = null;
			
	function openBaidu(){
		//打开百度窗口，并让obj指向百度窗口
		obj = window.open('http://www.baidu.com');
        //obj = open('http://www.baidu.com');       //window可以省略
	}

	function closeBaidu(){
		//关闭百度窗口
		obj.close();
	}
</script>

# 7.2.6 setTimeout()定时器

setTimeout()函数能够在指定的时间间隔后，调用一个函数（此函数称为回调函数）

注意：setTimeout()在指定的时间间隔后，只调用一次

示例： 
setTimeout(function(){
	console.log('喂喂，3秒到了');
},3000);

function callback(){
	console.log('hello，3秒到了');
}
setTimeout(callback,3000);

let callback = function(){
	console.log('hello，3秒到了');
}
setTimeout(callback,3000);

# 7.2.7 setInterval()定时器

setInterval()函数能够在指定的时间间隔后，调用一个函数（此函数称为回调函数）

注意：setInterval()在指定的时间间隔后，可以调用无限多次

示例：

在控制台中，每隔一秒钟，输出一个当前时间：

function getCurTime() {
	let now = new Date();
	let hour = now.getHours();
	let minute = now.getMinutes();
	let second = now.getSeconds();
	hour = hour < 10 ? "0" + hour : hour;
	minute = minute < 10 ? "0" + minute : minute;
	second = second < 10 ? "0" + second : second;
	return hour + ":" + minute + ":" + second;
}

let callback = function(){
	console.log(getCurTime());
}
setInterval(callback,1000);

# 7.2.9 location对象

location用于获取或设置地址栏中的url，并且提供若干方法用于刷新或跳转等。

  • location.reload() 页面重新加载（相当于刷新按钮）

  • location.href 返回或设置完整的url

示例： 
<button onclick="doReload()">刷新</button>
<button onclick="toBaidu()">跳转百度</button>
		
<script>
	function doReload(){
		location.reload();
	}
			
	function toBaidu(){
		location.href = 'http://www.baidu.com';
	}
</script>

# 7.2.10 history对象

history对象记录了用户曾经浏览过的页面(URL)，并可以实现浏览器前进与后退相似导航的功能。history对象就是访问后的URL。

示例：

创建两个html文件，第一个a.html文件中书写：

<a href="b.html">跳转b页面</a>

第二个b.html文件中书写：

<a href="javascript:history.go(-1)">回退</a>

# 7.2.11 navigator对象

Navigator 对象包含有关浏览器的信息。

  • userAgent属性：返回由客户端发送服务器的头部值。

  • appName属性：返回当前浏览器名称。

  • appVersion属性：返回当前浏览器版本信息。

  • platform属性：返回当前操作系统信息

示例： 
console.log(navigator.userAgent);
console.log(navigator.appName);
console.log(navigator.appVersion);
console.log(navigator.platform);

# 7.2.12 Screen对象

Screen 对象包含有关客户端显示屏幕的信息。

  • width属性：返回显示器屏幕的宽度。

  • height属性：返回显示器屏幕的高度。

示例： 
console.log(screen.width);
console.log(screen.height);

# 8. DOM模型

# 8.1 DOM模型简介

window对象下的document对象就是DOM模型。 DOM描绘了一个层次化的节点树，每一个节点就是一个html标签，而且每一个节点也是一个DOM对象。

# 8.2 操作DOM

# 8.2.1 获取DOM对象常用方法

获取DOM对象的常用方法有如下几种：

  1. getElementById() 通过元素的ID属性获取DOM对象，获取的是一个DOM对象。
  2. getElementsByTagName() 通过元素的标签名获取DOM对象，获取的是一组DOM对象。
  3. getElementsByClassName() 通过元素的class属性获取DOM对象，获取的是一组DOM对象。
  4. getElementsByName() 通过元素的name属性获取DOM对象，获取的是一组DOM对象。
示例： 
<p id="one">这是一个p标签</p>
<p class="first">这是一个p标签</p>
<p class="first">这是一个p标签</p>
<p name="username">这是一个p标签</p>
<p name="username">这是一个p标签</p>

<script>
	let oneObj = document.getElementById('one');
	oneObj.style.color = 'red';     //只有第一个p标签字体变红色
			
	let pArr = document.getElementsByTagName('p');
	console.log(pArr.length);       //获取所有p标签对象，是类数组对象，长度为5
			
	let firstArr = document.getElementsByClassName('first');
	console.log(firstArr.length);   //获取所有class为first的对象，是数组，长度为2
			
	let userArr = document.getElementsByName('user');
	console.log(userArr.length);    //获取所有name属性为user的对象，是数组，长度为2
</script>

# 8.2.2 其它获取DOM对象的方法

  1. document.documentElement 获取html对象
  2. document.body 获取body对象
  3. document.querySelector() 通过选择器获取一个DOM对象
  4. document.querySelectorAll() 通过选择器获取一组DOM对象
示例： 
<div>
	<p>这是一个p标签</p>
	<p>这是一个p标签</p>
</div>

<script>
	let htmlObj = document.documentElement;
	htmlObj.style.backgroundColor = 'red';       //设置html背景色
			
	let bodyObj = document.body;
	bodyObj.style.fontSize = '36px';             //设置body中所有元素的字体大小
			
	let pObj = document.querySelector('div p');
	pObj.style.color = 'blue';                   //第一个p标签有效
			
	let pArr = document.querySelectorAll('div p');
	console.log(pArr.length);                    //数组长度为2
</script>

# 9. DOM编程

# 9.1 事件

网页中的每个元素都可以产生某些事件。事件是可以被侦测到的行为。 每个元素都可以产生某些可以触发JavaScript函数的事件。比如：

  • 当用户单击按钮时，就发生一个鼠标单击（onclick）事件。

  • 当用户点击提交按钮时，就发生一个提交数据（onsubmit）事件。

javaScript中的常用事件：

# 9.1.1 javaScript中绑定事件方式

在javaScript中，给DOM元素绑定事件有三种方式：内嵌式、绑定式、监听函数式。

  1. 内嵌式事件方式：将javaScript事件直接写在html标签中
示例： 
<div onclick="add()">点我</div>
<script>
    //事件处理函数
	function add(){
		console.log('点击了div对象');
	}
</script>
  1. 绑定式事件方式：通过DOM对象，使用javaScript代码绑定一个事件
示例： 
<div>点我</div>
<script>
	let divObj = document.getElementsByTagName('div')[0];
    //function()匿名函数就是事件处理函数
	divObj.onclick = function(){
		console.log('点击了div对象');
	}
</script>
  1. 监听函数式事件方式：
示例： 
<div>点我</div>
<script>
	let divObj = document.getElementsByTagName('div')[0];
	divObj.addEventListener('click',function(){
		console.log('点击了div对象');
	},false);
</script>

# 9.1.3事件冒泡与事件捕获

事件冒泡就是指父元素和子元素有相同的事件, 当触发子元素事件时, 会向上冒泡, 同时也会触发父元素事件。

那么事实上，我们并不能确定：用户到底是想触发子元素事件，还是想触发父元素事件。

解决这个问题的思路是：设计两种事件流：

  1. 当用户触发了子元素事件时，先触发子元素事件，再触发父元素事件。这就是事件冒泡。（默认）

  2. 当用户触发

了子元素事件时，先触发父元素事件，再触发子元素事件。这就是事件捕获。

事件冒泡：

示例： 
<head>
	<style>
		div{
			width: 200px;
			height: 200px;
			background-color: red;
		}
		div p{
			width: 100px;
			height: 100px;
			background-color: blue;
		}
	</style>
</head>
<body>
	<div>
		<p></p>
	</div>
		
	<script>
		let divObj = document.getElementsByTagName('div')[0];
		let pObj = divObj.getElementsByTagName('p')[0];
			
		divObj.onclick = function(){
			alert('父元素的点击事件被触发');
		}
		pObj.onclick = function(){
		    alert('子元素的点击事件被触发');
		}
		
	</script>
</body>

阻止事件冒泡:

event.cancelBubble = true;

示例： 
pObj.onclick = function(event){
	alert('子元素的点击事件被触发');
    event.cancelBubble = true;
}

# 9.2 元素尺寸和位置

注意:

style尺寸和其它尺寸的区别：

  • style返回的是带 px 的字符串；offset返回的是纯数值；所以进行计算时，offset尺寸比较方便。

  • style是读写的；offset是只读的；

  • 使用style只能取得行内样式的值，offsetLeft则可以取得行内及非行内样式。

综上所述，获取元素尺寸与位置使用offset尺寸；设置元素尺寸与位置使用style尺寸；

示例： 
<div id="one"></div>

<script>
    let divBox = document.getElementById('one');

    //设置元素尺寸
    divBox.style.width = '300px';
    divBox.style.height = '200px';
    //设置元素位置
    divBox.style.position = 'absolute';
    divBox.style.left = '200px';
    divBox.style.top = '100px';

    //获取元素尺寸
    console.log(divBox.offsetWidth,divBox.offsetHeight);
    //获取元素位置
    console.log(divBox.offsetLeft,divBox.offsetTop);

    //获取可视窗口尺寸（html）
    console.log(document.documentElement.clientWidth,document.documentElement.clientHeight);
    //获取可视窗口中内容尺寸（body）
    console.log(document.body.clientWidth,document.body.clientHeight);
</script>

# 9.3.4 动态操作DOM对象

可以使用 javaScript动态的创建、插入或删除DOM节点。

示例： 
<div></div>

<script>
    let divObj = document.getElementsByTagName('div')[0];

    //创建新的a节点，追加到div中
    let aObj = document.createElement('p');
    aObj.appendChild(document.createTextNode('A'));
    divObj.appendChild(aObj);

    //创建新的b节点，追加到a节点之前
    let bObj = document.createElement('p');
    bObj.appendChild(document.createTextNode('B'));
    divObj.insertBefore(bObj,aObj);

    //删除子节点a
    divObj.removeChild(aObj);

        //创建新节点c，用来替换子节点b
    let cObj = document.createElement('p');
    cObj.appendChild(document.createTextNode('C'));
    divObj.replaceChild(cObj,bObj);
</script>

    <script>
        let name = document.querySelector('input[name="name"]');
        let age = document.querySelector('input[name="age"]');
        let edu = document.querySelector('input[name="education"]');
        let addBtn = document.getElementById('add')
        let table = document.getElementsByTagName('table')[0]
        addBtn.onclick = function () {
            let tr = document.createElement('tr')
            let nameTd = document.createElement('td')
            let ageTd = document.createElement('td')
            let eduTd = document.createElement('td')
            let btnTd = document.createElement('td')
            let btnInput = document.createElement('input')
            nameTd.innerText = name.value
            ageTd.innerText = age.value
            btnTd.innerText = edu.value
            btnInput.setAttribute('type', 'button')
            btnInput.setAttribute('value', '删除')
            btnInput.onclick = function () {
                table.children[0].removeChild(tr)
            }
            btnTd.appendChild(btnInput)
            tr.appendChild(nameTd)
            tr.appendChild(ageTd)
            tr.appendChild(eduTd)
            tr.appendChild(btnTd)
            table.children[0].appendChild(tr)
            name.value = ''
            age.value = ''
            edu.value = ''
        }
        function delClick(then) {
            // console.log(then.parentNode.parentNode);
            table.children[0].removeChild(then.parentNode.parentNode)
        }
    </script>
