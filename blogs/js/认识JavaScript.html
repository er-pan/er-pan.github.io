认识JavaScript及相关

  1. 脚本语言：JavaScript是一种解释型的脚本语言，可以直接执行，用于客户端、基于对象，不包括面向对象。
  2. 基于对象：JavaScript是一种基于对象的脚本语言,它不仅可以创建对象,也能使用现有的对象。
  3. 简单: 是弱数据类型语言，声明时不需要指定数据类型。数据类型完全由值类型决 定。
  4. 跨平台：JavaScript与操作系统无关，只要有一个js引擎（运行环境），就可以运行js代码。
  5. 嵌入式：需要在html页面上操作html元素，因此需要调用浏览器提供的操作html元素的接口（html dom接口）。

相关：

  • 1994年12月发布第一版
  • 1997年7月，ECMA组织发布第一版，规定浏览器脚本语言的标准，称为ECMAScript
  • 2015年6月，ECMAScript 6正式（ES6)，并更名为ECMAScript 2015
  • 国际化标准化组织：ECMA(European Computer Manufacturers Association)
  • ISO：国际标准

# javaScript组成部分

# 1. javaScript的三个主要组成部分是：

  1. ECMAScript(核心) --(European Computer Manufacturers Association)
  2. DOM（文档对象模型）--Document Objcet Model
  3. BOM（浏览器对象模型) --Browser Objcet Model

# 2. javaScript基本结构

# 2.1 javaScript基本结构

一个HTML页面中，javaScript代码可以写在script标签

代码示例 
<script type="text/javascript" language="JavaScript">
	//在控制台输出“hello world！”
	console.log('hello world!');
</script>

⚠️ 注意：

  • script标签中的type属性与language属性都可以省略。
  • 行尾可以写分号，也可以不写;
  • 单行注释：// 多行注释： /* */

# 2.2.javaScript在页面中的位置

  • 放在head标签
  • 放在body标签的最后位置
  • script标签可以有多个

# 2.3 网页中引用JavaScript的方式

可以把HTML文件和JS代码分开, 单独创建一个外部JavaScript文件

在HTML文件的head标签或body标签引入

::: 示例

<script src="./js/test.js"></script>

:::

⚠️ 注意：

在JS文件中，不需要script标签,直接编写JavaScript代码就可以了

建议放到body标签最后位置，若放到head标签位置，建议js代码中添加文档就绪函数 window.onload = function() { ... }

# 3. javaScript基本语法

# 3.1 变量的声明与赋值

使用let声明变量 语法： let 变量名 = 变量值；

示例 
let num = 10;
console.log(num);

::: warnnig ⚠️ 注意：

  • let是关键词，后面跟变量名。表示声明一个变量
  • 变量的数据类型由所赋值的类型来决定

:::

变量的命名规范:

  1. 变量必须使用字母、下划线( _ )或者美元符( $ )开始。
  2. 然后可以使用任意多个英文字母、数字、下划线或者美元符组成。
  3. 变量名区分大小写，如:A与a是两个不同变量。
  4. 不能使用JavaScript关键词与JavaScript保留字。
  5. 变量名最长为255个字符（实际上可以超过）。

**使用var声明变量 ** 语法： var 变量名 = 变量值；

示例 
var num = 10;
console.log(num);

# 3.2 js中的数据类型

js中的数据类型有：

  1. undefined-未定义类型：未被赋值的变量，值为undefined
  2. boolean-布尔类型：值为true/false
  3. number-数字类型：任何数字（NaN也是number类型，表示“非数”）
  4. string-字符串类型：值用引号（单引号或者双引号）括起来
  5. object-对象类型：null、数组、Date等
  6. function-函数类型：function函数
  7. symbol-ES6中新的原始数据类型，表示独一无二的值
  8. null-空数据类型

注意：以上数据类型中，undefined、boolean、number、string、symbol也被称为原始数据类型

示例 
let num;   
console.log(typeof num)       //返回undefined
num = 10;
console.log(typeof num)       //返回number
num = 'abc';
console.log(typeof num)       //返回string
num = true;
console.log(typeof num)       //返回boolean
			
function getDate(){}
console.log(typeof getDate)   //返回function
			
num = [];
console.log(typeof num)       //返回object
num = new Date();
console.log(typeof num)       //返回object
num = null;
console.log(typeof num)       //返回object
let s = Symbol();
console.log(typeof s)    	 // 返回"symbol"
var nll = null;

# 3.3 特殊值 NaN、Infinity

NaN，即非数值(Not a Number)是一个特殊的值

isNaN()函数在接收到一个值之后，会尝试将这个值转换为数值，如果可以转就返回false

isNaN()判断是不是 不是一个数字

示例 
console.log(parseInt('a'));      //NaN（parseInt()函数：将string转换成number）

console.log(isNaN(5));     //false
console.log(isNaN('5'));   //false
console.log(isNaN('a'));   //true

# 3.4 js中的运算符

javaScript中的运算符有五种：算术运算符、比较运算符、逻辑运算符、赋值运算符、三目运算符。

3.4.1 算术运算符

符号 说明
++ 自加一（递增。i++ 相当于 i = i + 1;） 注意：i++ （先用后加） ++i（先加后用）
-- 自减一（递减。i-- 相当于 i = i - 1;） 注意：i-- （先用后减） --i（先减后用）

++与 -- 实例：

示例： 
let num = 1;
console.log(num++);       //1
console.log(num++);       //2
console.log(num--);       //3
console.log(++num);       //3
console.log(num);         //3
console.log(--num);       //2

3.4.2 赋值运算符

符号 说明
= 赋值运算符
+= 加快捷赋值运算符； i += 1; 等效于： i = i + 1;
-= 减快捷赋值运算符； i -= 1; 等效于： i = i - 1;
*= 乘快捷赋值运算符； i *= 1; 等效于： i = i * 1;
/= 除快捷赋值运算符； i /= 1; 等效于： i = i / 1;

3.4.3 比较运算符

符号 说明
== 等于（运算符在比较前自动转换不同类型的值） 比如：1=="1" 结果为：true； null == undefined 结果为：true； 1==true 结果为：true
!= 不等于 比如：1!="1" 结果为：false
=== 严格相等（运算符不自动转换不同类型的值） 比如：1==="1" 结果为：false； null === undefined 结果为：false
!== 严格非相等（运算符不自动转换不同类型的值） 比如：1!=="1" 结果为：true
示例：

比较运算符在比较前会自动转换不同类型的值：

console.log(1==1);            //true
console.log(1=='1');          //true
console.log(1!='-1');         //true

严格比较运算符不自动转换不同类型的值，也叫等同运算符：

console.log(1===1);           //true
console.log(1==='1');         //false
console.log(1!=='1');         //true

3.4.4 逻辑运算符

符号 说明
&& “与”操作符，只有“&&”两边值同时满足(同时为真)，整个表达式值才为真。
|| “或”操作符，当两个条件中有任一个条件满足，运算结果就为“真”。
! “非/取反”操作符，也就是"不是"的意思，真即假，假即真。
示例： 
console.log(1 == 1 && 5 > 2);         //true
console.log(1 == 1 && 5 > 5);         //false

console.log(1 == 1 || 5 > 2);         //true --左边为真则不需要看右边
console.log(1 == 1 || 5 > 5);         //true
console.log(!(1==1));         //false

3.4.5 运算符优先级

  • 按操作数多少划分优先级： 一元操作符 > 二元操作符 > 三元操作符

  • 按运算类型划分优先级： 算术运算符 > 关系运算符 > 逻辑运算符 > 赋值运算符

3.4.6 js中的数据类型转换

  1. 隐式转换：减、乘、除可自动进行隐式转换
示例： 
let num1 = '10';
let num2 = '5';
console.log(num1-num2)    //5
console.log(num1*num2)    //50
console.log(num1/num2)    //2
  1. 强制转换：
  • parseInt() ：将字符串转换成整数。
  • parseFloat() ：将字符串转换成浮点数。
示例： 
let str1 = '10';
let str2 = '10.5'
console.log(parseInt('10')+parseFloat('10.5'));  //20.5

# 4. 流程控制语句

# 4.1 条件语句

条件语句主要有两类语法：

  • if 语句

  • switch 语句

4.1.1 if 语句语法：

示例： 
if(条件表达式){
    //条件表达式为true时执行的语句块（也叫if块）;
}
let score = 80;
if (score >= 60) {
    console.log('及格了');
}

if-else 语句语法：

示例： 
if(条件表达式){
    //条件表达式为true时执行的语句块;
}else{
    //条件表达式为false时执行的语句块;
}
let score = 80;
if (score >= 60) {
    console.log('及格了');
}else{
    console.log("不及格");
}

多重 if 语句语法：

示例： 
let score = 80;
if (score >= 90) {
    console.log("优秀");
} else if (score >= 80) {
    console.log("良好");
} else if (score >= 60) {
    console.log("中等");
} else {
    console.log("差");
}

4.1.2 switch 语句语法：

示例： 
switch(表达式){
    case 值1:
        //当值1与表达式的值匹配时执行这里
        break;
    case 值2:
        //当值2与表达式的值匹配时执行这里
        break;
    case 值3:
        //当值3与表达式的值匹配时执行这里
        break;  
    default:
        //当上面的值都不匹配时执行这里
        break;
}

switch语句中的break:

在switch语句中，一旦有一个case匹配，那么其它case将不在进行匹配，而是无条件向下执行。break关键词，当一个case执行完毕后，就依靠break语句退出整个switch结构。

let level = 2;  //比赛名次
switch(level){
    case 1:
        console.log("奖励奔驰小轿车");
        break;
    case 2:
        console.log("奖励华为手机");
        break;
    case 3:
        console.log("奖励泡泡糖一个");
        break;  
    default:
        console.log("奖励假期作业100本");
        break;
}

⚠️ 注意:

  1. 表达式的类型只能为byte、short、char、int、String；
  2. 值1、值2......值n只能为常数或常量，不能为变量，而且值不能重复；
  3. case可以写任意多个；
  4. break关键字结束switch语句，为可选项；
  5. default语句为可选项，当所有case都不匹配时执行default块。

4.1.3 if与switch的区别

  • if语句用于区间判断，switch语句用于等值判断。

  • switch语句书写的代码，都可以使用if替换；但if语句书写的代码，不一定能用switch替换。

# 4.2 循环语句

循环，需要具备三个要素：

说明：

  1. 循环变量（充当计数器的角色）
  2. 退出条件（用于结束循环）
  3. 循环变量的改变（计数器必须要改变，否则将无法结束循环）

不论是while循环语句，还是for循环语句，它们的本质都是实现了循环三要素。

4.2.1 while循环语法:

示例： 
//声明一个循环变量
let i = 0;
//退出条件判断
while(i<5){
    //这里是循环体
    console.log("hello");
    //循环变量的改变
    i++;
}

4.2.2 死循环:

示例： 
let i = 0;
//退出条件永远不成立，造成死循环
while(i>=0){
    console.log("hello");
    i++;
}

let i = 0;
while(i<5){
    console.log("hello");
    //循环变量没有改变，那么退出条件永远不成立，造成死循环
}

4.2.3 do while循环

需求：计算从1加到10的总和，但是如果总和大于30就停止累加

示例： 
let i = 1;
let sum = 0;
do {
    sum += i;
    i++;
}while(sum<=30);
console.log(sum);

4.2.4 while与do while的区别:

  • while：先判断，后执行。执行次数：0-N

  • do-while：先执行，后判断。执行次数：1-N

4.2.5 for 循环

for循环将循环三要素书写在一起:

示例： 
//for中的三个表达式就是循环三要素
for(let i=0;i<5;i++){
    console.log("hello");
}

for(;;){
    console.log("hello");
}

说明

执行过程：

在for后面的小括号中，使用两个分号分割成三个部分。

第一个部分的表达式用于声明循环变量并初始化。它在for语句中执行且只执行一次。

第二个部分的表达式是循环条件，所以此表达式的值必须为布尔类型。

第三个部分的表达式是循环变量变化的语句。

注意：for循环中的三个要素都可以省略，但两个分号不能省略。比如：下面是一个for形式的死循环

4.2.6 while与for的区别:

  • for循环结构紧凑简洁，常用于能够预先判断循环次数的循环中。

  • while结构松散，常用于无法事先判断循环次数的循环。

无本质上的区别, 两者能够完成的事情是一样的

4.2.7 break与continue

  • break：改变程序控制流。用于跳出当前switch或循环结构。

  • continue：改变程序控制流。只能用于循环中，用于跳出本次循环，然后执行下一次循环。
